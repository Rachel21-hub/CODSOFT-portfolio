# 🌟 CODSOFT Portfolio

A responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**.  
This project showcases your skills, projects, and profile in a clean and modern layout.

---

## 📌 Project Overview

The CODSOFT Portfolio is a static website that highlights personal or professional information.  
It demonstrates responsive design, use of sections (About, Projects, Contact), and visual styling.

---

## 🧩 Features

✔ Hero section with introduction  
✔ Skills / About section  
✔ Projects or Work section  
✔ Contact section or form  
✔ Responsive layout for desktop and mobile  
✔ Smooth navigation

---

## 🛠 Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript** (for interactive features)

No frameworks or libraries required (vanilla development).

## 📁 Project Structure

```
CODSOFT-portfolio/
```

- `index.html` — Main portfolio layout

## ▶️ How to Run

To view the portfolio:

1. Clone the repository:

```bash
git clone https://github.com/Rachel21-hub/CODSOFT-portfolio.git
```

2. Go inside the folder:

```bash
cd CODSOFT-portfolio
```

3. Open `index.html` in your browser:

```bash
start index.html   # Windows
```

Alternatively, simply double-click the file.

No installation necessary.
